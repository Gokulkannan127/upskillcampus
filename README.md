# 🏦 Banking Information System

A Python-based Banking Information System that simulates basic banking operations using Object-Oriented Programming (OOP). The application allows users to create bank accounts, perform transactions, and store account information using JSON for data persistence.

## 📌 Features

- ✅ Create New Bank Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- 📜 View Transaction History
- 💾 Automatic Data Storage using JSON
- 🔍 View All Accounts
- ⚠️ Input Validation & Error Handling

## 🛠️ Technologies Used

- Python 3.8+
- Object-Oriented Programming (OOP)
- JSON File Handling
- Datetime Module
- Visual Studio Code
